# `build-plan-specs` agent skill

Start from a detailed product description, and systematically turn that into an implementation specifications.

Here's a lightning talk, on what this is about:

[![](https://i.ytimg.com/vi/yZNkGqRLrPw/maxresdefault.jpg)](https://www.youtube.com/watch?v=yZNkGqRLrPw&list=PLjyCRcs63y804L55-eIJtrNxMBFbbroL8&)

## Setup

```shell
npx skills add -g bguiz/build-agent-skills --skill build-plan-specs
```

## Usage

Within your gen AI agent harness, invoke the skill, e.g.

```text
Help me turn my product design to into implementation specifications
```
